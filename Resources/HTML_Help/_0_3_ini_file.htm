INI File description of RS-ITS RunSequence

General

The ini-file RunSequence.ini should be saved in the install directory of RunSequence (default: "C:\Program Files\Spectral Electronic\SW-ITS\Bin\")

You can control RS-ITS with different tags in the ini file.

Control section and keys

The section name is RunSequence. Do not edit the section names and the key name. In case of change RS-ITS RunSequence do not work properly.

ShowFileSelectPopup

1: The user must select a sequence, that is stored in DefaultDirectory. The file select popup appears after start of RS-ITS RunSequence.

0: The file select popup do not appear.

Default value: ShowFileSelectPopup = 0

DefaultDirectory

Specify a directory that be show if ShowFileSelectPopup value is set to 1.

Default value: DefaultDirectory = "C:\Programme\Spectral Electronic\SW-ITS\Projects\"

QuitAfterExecution

1: If the execution of selected sequence ends, RS-ITS RunSequence quits automatically.

0: If the execution of selected sequence ends, RS-ITS RunSequence do not quit automatically.

Default value: QuitAfterExecution = 0

TracingEnabled

1: RS-ITS RunSequence save all actions in a tracing file that would be stored in install directory.

0: No trace file will be created.

Default value: TracingEnabled = 0

ShowLoadAndRunMenu

1: The menu item File -> Load and run sequence is enabled.

0: The menu item File -> Load and run sequence is disabled.

Default value: ShowLoadAndRunMenu = 1

EnableFixtureCheck

1: The button for activating fixture code check is enabled.

0: The button for activating fixture code check is disabled.

Default value: EnableFixtureCheck = 1

AutoStartFixtureCheck

1: The fixture code check process starts automatically after RS-ITS RunSequence was started. All resources of the GUI are disabled.

0: The fixture code check process does not start automatically after RS-ITS RunSequence was started.

Default value: AutoStartFixtureCheck = 0

NoFixtureCode

Specify a fixture code, that was returned by the GetFixtureCode.dll, if no fixture was mounted on test system.

Default value: NoFixtureCode = ""

FixtureSureCounter

The fixture code check process gets the fixture code one time per second. It is possible that the fixture code, while mounting the fixture, not recognized correctly. The value determines how often the code is read, before it is valued.

Default value: FixtureSureCounter = 3

RunGlobalBatchFile

Specify a batch file (incl. the complete path). This file is executed before the TestStand Engine is initialized. Use this option to start synchronization processes.

Default value: RunGlobalBatchFile = ""

WaitForGlobalBatchFile

0: The batch file is started without waiting for the end of the execution.

1: The batch file is started and waits for the end of the execution.

Default value: WaitForGlobalBatchFile = 1

DelayAfterGlobalBatchFile

Does not wait for the end of the execution of the batch file, you have the opportunity here to set a fixed waiting time.

Default value: DelayAfterGlobalBatchFiler = 0.0

Example:

[RunSequence]
ShowFileSelectPopup = 0
DefaultDirectory = "C:\Programme\Spectral Electronic\Sw-ITS\Projects\"
QuitAfterExecution = 0
TracingEnabled = 0
ShowLoadAndRunMenu = 1
EnableFixtureCheck = 1
AutoStartFixtureCheck = 0
NoFixtureCode = ""
FixtureSureCounter = 3
RunGlobalBatchFile = ""
WaitForGlobalBatchFile = 1
DelayAfterGlobalBatchFile = 0.0

Product sections and keys

RS-ITS RunSequence has a sequence file window. You can specify product names or ident numbers with corresponding sequence file and optional a fixture code. You can start test programs manually from sequence file window (hit "Run Selected Sequence" button) or automatically by fixture codes ("Fixture Code Check" button).

 

Create a new section in RS-ITS RunSequence INI file to specify product and sequence file in the sequence file window. The section name appears in the sequence file window. Specify the following keys in the new section:

FixtureCode

The fixture code must be an bijective value to identify the section. Please use quotes for this key!

Sequence

Specify the sequence path and file for your product and corresponding fixture code. Please use quotes for this key!

RunBatchFile

Specify a batch file (incl. the complete path). This file is executed before the selected sequence is executed. Use this option to start synchronization processes. Please use quotes for this key!

WaitForBatchFile

0: The batch file is started without waiting for the end of the execution.

1: The batch file is started and waits for the end of the execution.

DelayAfterBatchFile

Does not wait for the end of the execution of the batch file, you have the opportunity here to set a fixed waiting time.

Example:

[3524-S-29898-2]
FixtueCode = "101"
Sequence = "C:\Prüfprogramme\3524-S-29898-2\Testprogramm.seq"

[3524-S-29989-1]
FixtueCode = "102"
Sequence = "C:\Prüfprogramme\3524-S-29989-1\Testprogramm.seq"
RunBatchFile = "C:\run_sync_proc.bat"
WaitForBatchFile = 1
DelayAfterBatchFile = 0
