Command line arguments

RS-ITS RunSequence accept the following command line arguments:

/shellmode

This arguments change the exit behaviour of RS-ITS RunSequence. The menu option "File -> Quit" change to "File -> Quit and Logoff". RS-ITS RunSequence initiate a logoff process in operating system.

/trace

This argument enables the tracing mode (see here). The argument overwrite the ini file option.

/file:C:\Program Files\Spectral Electronic\SW-ITS\Projects\Testprogram.seq

Pass this argument to run a sequence immediately after start up RS-ITS RunSequence.
