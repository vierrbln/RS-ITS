General

RunSequence (RS-ITS) is based on National Instruments Teststand and uses the engine of NI Teststand. You should use the program to start and control your test programs (NI Teststand sequence files) that you are developed with NI Teststand and the Spectral Test Libraries TL-ITS. This tool can do:

  • Start sequence executions
  • Terminate sequence executions
  • Abort sequence executions
  • Load und run sequence files automatically in order to a defined fixture code

Login Process

RS-ITS uses den engine of NI Teststand, thus the user manager of NI Teststand. If you start RS-ITS, you have to login with your user name and password. RS-ITS checks the privilige to operate sequences. For more information please refer to the Teststand manuals.

Loading and Running Sequences

The easiest way to start a sequence file is to start RS-ITS and hit the Menu "File -> Load and RunSequence". Select a sequence and click the "Load" button. The execution starts immediately.

Hint
We recommend that RS-ITS only executes templates sequences of SW-ITS with standard callback sequences, e.g. SequenceFileUnload. RS-ITS checks for these callback and shows you a confirm popup.
 

Quit a execution

To end a execution of standard SW-ITS Teststand sequences you should use the "File -> Quit" menu item comes with CP-ITS (Control panel). The execution ends programmitically. This is the best way to end a execution. RS-ITS receive the end event and resets all reosurces of the user interace. It is possible that the sequence never ends (e.g. bug in sequence file or wrong sequence model). In this case you have to try two options:

  1. Terminate execution: Click the menu item "Debug -> Terminate Execution". The Teststand engine stops the current execution and runs all cleanup routines specified in sequence file. This way is save to reset the test system (power off etc.). If you use standard sequence model, Teststand shows you two options: "Next UUT" or "Terminate". Please refer to Teststand manuals.
  2. Abort execution: Clickthe menu item "Debug -> Abort Execution". The Teststand engine try to stop the current execution immediately. Caution: Teststand don't run any reset sequences. All enabled sources in the test system may be leave activated!

Run a sequence from sequence window

If sequence files are specified in ini-file of RunSequence, RS-ITS will shows these files in the sequence file window. Click in these list to mark a sequence and hit the "Run Selected Sequence" button or press F5 to start the execution. Right mouse click on list items shows detailed information incl. fixture code.

 

Hint
Improve your execution performance: disable tracing and disable result recording. If you start RS-ITS and run a sequence, RS-ITS checks the internal settings and ask for disabling these options.
