Important information about RS-ITS RunSequence

To work with this tool, you need the following software packages:

  • National Instruments CVI runtime library 8.5 or higher (comes with the distribution kit)
  • National Instruments Teststand 4.0 or higher (Base deployment license, debug deployment license or development license)
